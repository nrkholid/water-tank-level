# Water-Tank-Level
Design of Monitoring and Controlling Water Tank Level using PHP Native and NodeMCU

Component :
1. NodeMCU ESP8266 LoLin V3
2. Expansion Board of NodeMCU LoLin V3
3. HC-SR04 Sensor
4. Relay 2 Channel
5. Water Pump
6. Jumper Wire

Software :
1. Arduino IDE
2. Visual Studio Code
3. XAMPP (Apache & MySQL)
